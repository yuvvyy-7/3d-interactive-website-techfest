# 🚀 TECHFEST — 3D Interacive Page

A modern and interactive landing page concept for **Techfest**, built using **HTML, CSS, and Vanilla JavaScript**. The website features a futuristic design with smooth animations, an interactive 3D cube, animated flip cards, scroll-triggered effects, and responsive layouts to create an engaging user experience.

## Features

* 🎨 Futuristic UI with orange and cyan accents
* 🧊 Interactive 3D rotating cube
* 🃏 Animated flip cards for event domains
* 📈 Scroll progress indicator
* ✨ Scroll-triggered animations
* 🔢 Animated statistics counters
* 📱 Fully responsive design

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla)
